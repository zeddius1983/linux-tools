## antigravity

Google's AI CLI tool (similar to Claude Code and Codex CLI). Installed from the official Google installer at `antigravity.google`. Exported to the host as `agy`.

## Implementation

**Base image**: `docker.io/ubuntu:24.04`

**Install**: Official installer script pulled from `antigravity.google/cli/install.sh`, placed at `/usr/bin`.

**Dependencies**: `curl`, `ca-certificates`, `git`, `jq`, `gh` (GitHub CLI — installed from the official GitHub APT repo).

**Export**: `bin:agy` + `desktop:agy:Antigravity CLI`

## Pitfalls and notes

- The installer is fetched at image-build time — rebuild the image to pick up new releases.
